# -
提供常用一键加解密及进制转换等功能
### 加解密工具盒子

-----

一个基于python+tkinter实现的加密解密工具箱GUI程序，目前只包含常用加解密功能，持续更新中



#### 环境需要

---

- Python
  - Python 3.6+
- Windows(32位、64位)
  - Windows XP及以上



### 使用方法

----

下载`加解密工具盒子.exe`，双击运行即可

源码存入文件夹`Lib`下，通过pyinstaller打包为exe，放心使用
